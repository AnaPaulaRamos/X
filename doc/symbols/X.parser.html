reslice(object, datastream, sizes, min, max)
Reslice a data stream to fill the slices of an X.volume in X,Y and Z directions. The given volume (object) has to be created at this point according to the proper dimensions. This also takes care of a possible associated label map which has to be loaded before.
Parameters:
{!X.object} object
The X.volume to fill.
{!Array} datastream
The datastream as an array.
{!Array} sizes
The sizes of the volume as an array [X,Y,Z].
{!number} min
The min. scalar intensity value.
{!number} max
The max. scalar intensity value.
